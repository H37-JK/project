### 1. 왜 `uv`를 추천하나요? (압도적 장점)

*   **진짜 빠릅니다:** Rust로 만들어져서 기존 `pip`나 `poetry`보다 **10~100배 정도 빠릅니다.** 라이브러리 설치할 때 기다리는 시간이 거의 없습니다.
*   **올인원(All-in-one) 도구:**
    *   파이썬 버전 관리 (`pyenv` 역할)
    *   가상환경 생성 및 관리 (`venv` 역할)
    *   패키지 설치 및 의존성 해결 (`pip`, `pip-tools` 역할)
    *   프로젝트 관리 (`poetry` 역할)
*   **Docker와 찰떡궁합:** 설치 속도가 워낙 빨라서 Docker 빌드 시간을 획기적으로 줄여줍니다.
*   **단일 실행 파일:** 파이썬이 설치되어 있지 않아도 `uv`만 설치하면 알아서 파이썬까지 깔아줍니다.
